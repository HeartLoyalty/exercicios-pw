Primeiros passos do Git

  1. Instale o Git na sua Maquina pelo site oficial deles disponivel em Git install
  2. Após o Git instalado Crie um Diretorio no seu computador que vai ser onde irá deixar todas as suas pastas de trabalho que você vai subir para o Git Hub.
  3. Nesse diretório voce poderá adicionar e excluir quantos arquivos vc quiser, coloque neles quantos trabalhos puder fazer(mas deixando claro uma coisa aqui, todos esses arquivos ficaram em apenas 1 Repositório.) para Criar Multiplos Repositórios vc deve Ter um Diretório Raiz para armazenar as pastas e arquivos.
